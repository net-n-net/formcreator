What does it do?
================
Here are some benefits of this extension.

**Best practise**

- Forms are created in the frontend
- Easy to add new fields with drag&drop function
- Define your own validations provided by https://www.formvalidator.net
- Automatically logs all submitted values to database
- Export logs in the TYPO3 backend
